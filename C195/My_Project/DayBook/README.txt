	TITLE and PURPOSE  
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	The "DayBook" is  developed with the intension of managing records of Customers and their Appointments.
	Using DayBook, User can  perform basic data operation : add/edit/delete customer and appointments is a safe way. 
	DayBook follows the modern UI design patterns. The Advance data validation proces of DayBook can benifit user
	saving countless hour of manpower hours and boosters the data accuracy. The DayBook can generate three different
	types of customized reports based on user's selections.

	Directions - how to run the program ?
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	* Launch the Application : enter valid username password (username: test - password: test) or (username: admin - password: admin) 
	* valid Login navigated you to the Main UI :

		- If user have any appointment with in 15 minutes of login user will get the alert for each of those appointments
		- Main UI natigates users to 3 sub - ways (Appointments, Customers, Reports)

	* Appointments UI   : allows user to view all appointsments , naviagtes user to ADD, MODIFY UI forms and allows to DELETE the appointments records.
	* Customer UI 	    : allows user to view all cusomers , naviagtes user to ADD, MODIFY UI forms and allows to DELETE the customers records.
	* Reports UI        : allows user to generate following reports by making desired selections

				1. Appointments by Month and Type
				2. Appointments by Contact's Schedule
				3. Customers by Country


===================================================================================================================================================================================
	Description : Report 3 (Customers by Country)
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
	
	The Report type 3 allows user to view the customers based on country of their record. There is a Drop Down box (ComboBOX) where user can 
	select one of the three countries (US, UK, CANADA) and view the customes reside in any of these specific countries.
